# NgeLinkDo

### Modern & Static 'Link in Bio' Generator

NgeLinkDo is a fast, visually appealing, and highly customizable 'Link in Bio' page generator. Tailor your online identity, list all your essential links, and export them into a single, lightweight HTML file that you can host anywhere for free.

## Key Features

- Modern UI/UX: Built with a clean interface, incorporating Glassmorphism effects and smooth transitions.
- Fully Responsive: Looks perfect on both desktop and mobile screens.
- Dark/Light Mode: Seamlessly switch between themes based on user preference.
- Static Export: No database or backend required. Generate a single index.html file containing your entire customized page and host it on platforms like Netlify, GitHub Pages, or Vercel.
- Localized: User-friendly interface tailored for Indonesian users.

## Tech Stack

- Frontend: React.js
- Styling: Tailwind CSS
- Icons: React Icons

## Quick Start

To get this project up and running on your local machine, follow these steps:

1. Clone the repository:

   ```bash
   git clone [https://github.com/yourusername/ngelinkdo.git](https://github.com/yourusername/ngelinkdo.git)

   ```

2. Navigate to the project directory:

   ```bash
   cd ngelinkdo

   ```

3. Install dependencies:

   ```bash
   npm install

   ```

4. Start the development server:
   ```bash
   npm start
   ```
